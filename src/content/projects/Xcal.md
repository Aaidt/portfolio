- Engineered a canvas-based drawing engine using the Canvas API supporting 5 different shape
types, transforms, and freehand paths.
- Enabled real-time collaboration via WebSockets with room-based state syncing, supporting 10+
concurrent users per room.
- Created a whiteboard-style canvas using RoughJS and math-driven logic for freehand pencil
strokes.
- Optimized redraw and event-handling logic to maintain 45 FPS during multi-user interactions.
